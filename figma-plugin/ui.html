Pixel-Lock Exporter

导出当前选中 Frame；如果没有选中，则导出当前页面所有顶层 Frame。

新手路径：先在这里导出，再让 AI 读取本地 manifest/assets 写代码。这个通道不走 Figma REST API token，也不消耗官方 MCP 调用额度。
1终端启动 Bridgenpm run plugin-bridge
2选中画板后导出先检查 Bridge，再点击开始导出。
3把提示词交给 AIAI 读取本地文件后生成前端。
先在终端运行：npm run plugin-bridge
